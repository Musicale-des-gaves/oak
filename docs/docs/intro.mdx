# Introduction

⚡️ Oak will let you build anything from a webpage to an email, or even paywalls (that's what we do [@poool](https://poool.tech)).

Don't forget to import styles, for example using `style-loader` and `webpack`:

```javascript
import '@poool/oak/dist/oak.min.css';
```

Or import them directly inside your own styles using `less`, `sass` or `stylus`:

```css
@import "@poool/oak/dist/oak.min.css";
```

## Using without React

Oak is built on top of React. By default, React is bundled inside oak so you don't have to do it yourself.
If you're already using React in your application, you can import the standalone version (which still needs React but refers to it as an external dependency):

```javascript
import { render } from '@poool/oak/dist/standalone/esm';
```
